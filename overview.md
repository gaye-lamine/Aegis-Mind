# Splunk Agentic Ops Hackathon Overview

Reimagine the future of agentic operations using Splunk AI.

## Quick Facts
*   **Deadline:** June 15, 2026 @ 5:00 PM GMT+1 (9:00 AM PDT)
*   **Location:** Online & Public
*   **Prize Pool:** $20,000 USD in Cash + .conf26 Passes
*   **Hosting Platform:** Managed by Devpost
*   **Themes:** Machine Learning/AI, Cybersecurity, Enterprise observability

## About the Challenge
The Splunk Agentic Ops Hackathon invites developers, security, IT, and network engineers, and all AI-forward builders to create innovative solutions that combine AI with the power of Splunk. 

Participants will build intelligent applications that enhance observability, security/network operations, and developer productivity using the latest Splunk AI capabilities.

During the hackathon, teams will develop projects across three core tracks: **Observability**, **Security**, and **Platform**, while leveraging cutting-edge capabilities such as:
*   AI agents for Splunk apps
*   Splunk MCP Server
*   Splunk hosted models
*   AI Assistant
*   AI-powered app development tools

## Three Core Tracks

### 📊 Observability
Build solutions that help engineering teams (software, ITOps, or NetOps) better understand system behavior, detect anomalies earlier, and automate operational responses using AI.

### 🛡️ Security
Build solutions that help security teams detect threats faster, investigate incidents more efficiently, and automate security workflows using AI and Splunk data.

### ⚙️ Platform & Developer Experience
Build solutions that enhance the developer experience, automate workflows, or simplify how applications interact with Splunk data and APIs.

## Required Deliverables
1.  **Text Description:** Explaining project features and functionality.
2.  **Demo Video:** Under 3 minutes showing target device deployment, AI integration, and core value.
3.  **Public Open-Source Repo:** Detectable open-source license, full source code, and comprehensive `README.md` with dependencies and setup guide.
4.  **Architecture Diagram:** Stored as `architecture_diagram.(md|pdf|png)` at the root showing Splunk interaction, AI agent integration, and service data flows.

## Core Judging Criteria
*   **Technological Implementation (25%):** Quality software development.
*   **Design (25%):** Well thought out user experience and layout aesthetics.
*   **Potential Impact (25%):** How big of an impact the solution can make.
*   **Quality of the Idea (25%):** Creativity and uniqueness.
